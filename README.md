# learning_ml_dl_cv
В данном репозитории содержатся ноутбуки по:
- ML Школы21 в каталоге ML_S21
- DL МГУ в каталоге DL_MSU
- CV МГУ в каталоге CV_MSU

Второй каталог разделён на 2 подкаталога, в total - материалы общего курса ВМК по DL, в каталоге MMP - материалы [углубленного кафедрального курса кафедры ММП по DL](https://github.com/mmp-practicum-team/mmp_dl_spring_2025?tab=readme-ov-file)

В последнем представлены ноутбуки, выполненные в рамках университетского курса по классическому компьютерному зрению и работе с библиотекой OpenCV, включая итоговый проект, в котором реализовано детектирование транспортных средств на шоссе и их последующая классификация (неработающая из-за неудачно выбранного / отсутствия  подходящего датасета для обучения классификатора), после решил задачу классификации по направлению движения

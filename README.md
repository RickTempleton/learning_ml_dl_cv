# learning_ml_dl_cv
В данном репозитории содеражатся ноутбуки по
- ML Школы21 в каталоге ML_S21
- DL МГУ в каталоге DL_MSU
- CV МГУ в каталоге CV_MSU


В последнем представлены ноутбуки, выполненные в рамках университетского курса по классическому компьютерному зрению и работе с библиотекой OpenCV, включая итоговый проект, в котором реализовано детектирование транспортных средств на шоссе и их последующая классификация (неработающая из-за неудачно выбранного / отсутсвия подходящего датасета для обучения классификатора)
